# Rock Paper Scissors

A simple Rock Paper Scissors game built for [The Odin Project Foundations course](https://www.theodinproject.com/lessons/foundations-rock-paper-scissors).

This project focuses on basic JavaScript logic, functions, conditionals, user input, random computer choices, and score tracking. The first version of this project is played entirely in the browser console, as required by the lesson.

## Features

* Play Rock Paper Scissors against the computer
* Random computer choice generation
* Human player input
* Single-round winner logic
* Score tracking for the human player and computer
* Five-round game flow
* Console-based output

## Built With

* HTML
* JavaScript

## What I Practiced

* Writing JavaScript functions
* Using `Math.random()`
* Working with conditionals
* Comparing player choices
* Updating score variables
* Logging results to the console
* Breaking a problem into smaller steps

```bash

```

2. Open the project folder:

```bash
cd YOUR-REPO-NAME
```

3. Open `index.html` in your browser.

4. Open the browser developer tools.

5. Go to the Console tab to play the game.
